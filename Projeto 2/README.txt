Q-Digest
Para a execução o comando deve ser de tal forma:
$python qdig.py --val val --eps eps --univ univ data_file.csv rank/quant [query.args | --in query_args_file]
Para saber mais utilize o comando -h | --help

GK
Para execução o comando deve ser de tal forma
python gk.py --v (valor inteiro) --e (valor de eps desejado) --t [rank | quant] --f (path do arquivo) [--p | --i]
